# 众美高三物理题答工作流 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 接入众美高三物理 87 份题目与 87 份答案，使题目录入忽略可变分值图片段，答案按原始富文本安全提取并通过审核门禁。

**Architecture:** 题目录入继续使用理科共享规则，只在理科配置中增加通用的方括号分值提示正则，不绑定固定数字或图片哈希。答案侧新增专项批处理工具：整批只读预检并严格配对，按答案标记出现顺序连续编号，复制 OOXML 富文本与公式，移除题干和训练标题；富文本答案按既有物理规则转入解析承载。工作台仅补充“【空白试卷】”前缀归一化和本项目路径配置。

**Tech Stack:** Python 3、python-docx、lxml、OOXML/ZIP、WPS COM、unittest/pytest。

### Task 2: 众美物理答案富文本清洗器

**Files:**
- Create: `tools/clean_zhongmei_physics_answers.py`
- Test: `test_zhongmei_physics_workflow.py`

1. 写失败用例：完整题干中只保留 `答案/解析` 块；解析续行完整；编号重置按出现顺序连续化；内联 `答案 A解析 ...` 可拆分；缺解析补 `解析： `；源文件哈希不变；图片和公式关系保留。
2. 实现整批预检：仅处理 87 个原答案，剔除 `_已清洗`，题目名去掉 `【空白试卷】` 后严格配对；任何未知标记、数量不一致或无法切块都在写文件前停止。
3. 实现 OOXML 复制：选择题输出 `题号．答案`；普通大题输出 `题号． / 答案： / 解析：`；答案段含公式或对象时答案框留空，并把原答案富文本追加到解析中。
4. 为每份清洗结果生成审核清单和审核状态；审核清单放到 E 盘项目产物目录，状态文件与清洗答案同目录。
