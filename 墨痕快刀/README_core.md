## 🌟 核心优势

1.  **🛡️ 零修改 (Read-Only)**：
    - 脚本**只读不写**，绝对不会删除文档里的任何文字、图片或格式。
2.  **🧠 智能分科 (Smart Subject)**：
    - 自动识别文档科目（英语 / 理科 / 文科），加载专属规则，**三套配置互不干扰**。
    - **英语特供**：完美支持完形填空、七选五、阅读理解（A/B篇）、写作、大题提能、听力材料等特殊题型的**整体打包录入**。
    - **语文/文科特供**：支持 `(一)阅读下面…` 类大题块的**模块打包**，自动吞并所属小题，且遇到大题头（如 `四、写作`）自动阻断，不跨区合并。
3.  **🧬 智能吞并 (Smart Merge)**：
    - **贪婪模式（英语）**：遇到 `完形`、`七选五`，自动吞并所有小题。
    - **绅士模式（英语）**：遇到 `维度`、`单句`，只吞并第 1 题，后续独立。
    - **超级模式（英语）**：遇到 `【大题提能】`，自动吞并罗马标题和第一篇文章。
    - **模块打包（文科）**：遇到 `(一)(二)` 类VIP标题，自动吞并下属所有小题，obstacle边界自动阻断。
4.  **🔄 多文档热切换**：
    - 录完一科想换下一科？关掉文档，打开新书，在脚本里按 `r` 就能直接切换。

## 🛠️ 安装与准备

### 1. 环境要求
- Windows 系统
- Python 3.x
- WPS Office 或 Microsoft Word

### 2. 安装依赖库
打开命令行 (Win+R -> cmd)，运行：
```bash
pip install pywin32 pyautogui
```

### 3. 插件准备
- 打开 WPS 文档。
- 必须先打开【墨痕题库工具】的侧边栏（确保按 F1 有反应）。

## 🚀 操作流程

1. **启动脚本**：
   ```bash
   python main.py
   ```
   脚本会自动识别科目并显示章节列表。

2. **选择章节**：
   - 输入 `1`：录入第 1 章。
   - 输入 `1-3`：连续录入前 3 章。

3. **英语特殊题型处理**：
   - 单选题：自动逐题录入。
   - 阅读理解：按 A 篇、B 篇分别打包录入。
   - 完形/七选五：整道大题打包录入。
   - 写作：整段打包录入。
   - 大题提能：连同标题、原文、题目一起打包录入。

4. **语文/文科题型处理**：
   - 小题：逐题录入。
   - `(一)阅读下面…` 类大题块：自动打包该块下所有小题。
   - 遇到 `一、` `二、` `三、` 等大题头自动跳过（作为区域边界，不录入）。

5. **切换文档**：
   - 关闭当前文档，打开新文档。
   - 在脚本菜单里输入 `r` (Reload) 并回车。

## ⚙️ 高级配置 (Configuration)

### 1. 调试模式 (DEBUG_MODE)
在 `config.py` 中设置 `DEBUG_MODE = True`，脚本会在控制台打印详细的节点发现、吞并决策和过滤日志。

### 2. 添加路障 (obstacles)
如果你发现录入了不需要的文字，请在 `config.py` 中对应科目的 `obstacles` 列表添加正则：
```python
r"^我的新干扰项.*",
```

### 3. 学科自动识别规则
脚本通过文件名关键词 + 文档内容嗅探自动识别科目：
- **英语**：文件名含 `英语/English/外语/XB`，或内容含 `UNIT/Passage`
- **理科**：文件名含 `物理/化学/生物/科学/理综`
- **文科**：文件名含 `政治/历史/地理/道德/文综/语文`，或内容含 `现代文阅读/文言文/通假字` 等

## ❓ 常见问题

**Q: 为什么"大题提能"把 A 篇也吃进去了？**
A: 这是设计的特性。为了保证题目的完整性，大标题会包含它下面的第一篇文章。B 篇会作为第二题独立录入。

**Q: 为什么"维度一"只吃了一道题？**
A: 因为它属于"绅士 VIP"，只负责带个头，后面的题目需要独立录入以便于学生作答。

**Q: 语文的 `(一)阅读下面…` 为什么把好多小题都吃了？**
A: 这是文科模块打包模式。`(一)` 类标题会自动吞并其下属的所有小题，直到遇到下一个 `(二)` 或大题头边界。

**Q: 换文档后报错？**
A: 请按 `r` 键刷新目录。
